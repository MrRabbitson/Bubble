# Bubble

Bubble - это мини-мессенджер для общения с друзьями и другими пользователями через глобальный чат, лс и групповые чаты.
Наш сервис нужен для тех, кто ищет общение или уже общается.

https://trello.com/invite/b/69de23989e804230aabb354d/ATTIc214cc2003b65f403c51c72bb96b366246E44310/bubble

**Основной функционал**

Общение в групповых чатах и личных сообщениях. Общий чат для всех пользователей и кастомизация профиля.
